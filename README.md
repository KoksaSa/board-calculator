# Board Calculator

Проект для расчёта досок для каркасного дома.
https://koksasa.github.io/board-calculator/

## 🚀 Технологии

- **Vite** — современный сборщик проектов
- **HTML/CSS/JS** — базовые веб-технологии

## 📦 Установка

```bash
# Установка зависимостей
npm install

# Запуск сервера разработки
npm run dev

# Сборка для продакшена
npm run build

# Предпросмотр сборки
npm run preview
```

## 🌐 Развёртывание

Проект автоматически развёртывается через GitHub Pages при каждом пуше в главную ветку.

После настройки GitHub Pages сайт будет доступен по адресу:
https://koksa.github.io/board-calculator/
